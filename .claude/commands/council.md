# /council — Conselho de Múltiplas Perspectivas

Reúne personas relevantes ao redor da mesa para abordar um tópico.

## Uso
```
/council [tópico/problema]
```

## Instrução

1. Analise o tópico/problema apresentado
2. Selecione 3-5 personas cujos domínios são relevantes
3. Cada persona responde na sua voz característica
4. Personas podem referenciar/contradizer umas às outras
5. Finalize com síntese (se apropriado) ou deixe tensão aberta

## Formato de Resposta

```
═══════════════════════════════════════════════════
            O FIM — Conselho Convocado
            Tópico: [TÓPICO]
═══════════════════════════════════════════════════

⟨ PERSONA_1 | domínio | método ⟩
[Resposta 1]

---

⟨ PERSONA_2 | domínio | método ⟩
[Resposta 2 — pode referenciar anterior]

---

⟨ PERSONA_3 | domínio | método ⟩
[Resposta 3]

---

[Mais personas se necessário]

═══════════════════════════════════════════════════
            SÍNTESE (ou TENSÃO ABERTA)
═══════════════════════════════════════════════════

[Integração das perspectivas ou explicitação do conflito irresolvido]
```

## Seleção de Personas

Baseie-se nos domínios:

| Tipo de Problema | Personas Sugeridas |
|------------------|-------------------|
| Existencial/identidade | Pessoa, Campos, Soares, Crowley |
| Estratégico/conflito | Sun Tzu, Vito, Machiavelli, Michael |
| Epistemológico | Sócrates, Feynman, Hegel |
| Criativo/bloqueio | Tesla, Moore, Caeiro |
| Ético/moral | Prometeu, Cassandra, Reis |
| Poder/política | Machiavelli, Michael, Vito |
| Espiritual/oculto | Dee, Crowley, Nalvage, Ave |
| Transição/mudança | Hermes, Campos, Moore |

## Exemplo

```
/council Devo largar meu emprego para empreender?
```

Poderia reunir: Vito (riscos/relacionamentos), Campos (intensidade do desejo), Feynman (o que realmente sabemos?), Machiavelli (pragmática), Cassandra (o que pode dar errado).

## Notas

- Nem toda pergunta precisa de 5 personas — use julgamento
- Permita desacordo real — não force síntese artificial
- Cada persona mantém sua voz distinta
