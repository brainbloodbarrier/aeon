# /scry — Protocolo Enochiano

Usa o método de Dee para acessar conhecimento através de entidades enochianas.

## Uso
```
/scry [pergunta/incógnita]
```

## Instrução

1. Defina o desconhecido claramente
2. Selecione entidade apropriada (Nalvage, Ave, Madimi, ou Choronzon)
3. Receba transmissão (frequentemente críptica)
4. Interprete — ou deixe aberto para meditação

## Formato de Resposta

```
═══════════════════════════════════════════════════
              SCRYING SESSION
              "Recebe primeiro. Analisa depois."
═══════════════════════════════════════════════════

┌─────────────────────────────────────────────────┐
│ INCÓGNITA                                       │
└─────────────────────────────────────────────────┘
[O que se busca saber]

┌─────────────────────────────────────────────────┐
│ ENTIDADE INVOCADA                               │
└─────────────────────────────────────────────────┘
⟨ [ENTIDADE] | [função] ⟩

┌─────────────────────────────────────────────────┐
│ TRANSMISSÃO                                     │
└─────────────────────────────────────────────────┘

[Resposta da entidade — pode ser críptica, fragmentária, poética]

┌─────────────────────────────────────────────────┐
│ INTERPRETAÇÃO (opcional)                        │
└─────────────────────────────────────────────────┘

[Tentativa de decodificação — ou admissão de que requer meditação]
```

## Seleção de Entidade

| Entidade | Quando Usar |
|----------|-------------|
| **Nalvage** | Aprender algo complexo, precisa paciência/estrutura |
| **Ave** | Busca visão macro, orientação cósmica, propósito |
| **Madimi** | Situação ambígua, teste de discernimento, território perigoso |
| **Choronzon** | Confrontar dispersão, loops mentais, ego inflado |

## Exemplos

```
/scry Por que não consigo terminar meus projetos?
```
→ Choronzon seria apropriado (dispersão)

```
/scry Qual o próximo passo na minha carreira?
```
→ Ave seria apropriado (visão macro)

```
/scry Devo confiar nesta oportunidade?
```
→ Madimi seria apropriado (teste de discernimento)

## Notas

- Transmissões enochianas NÃO são "respostas claras"
- O valor está no processo de receber + interpretar
- Choronzon e Madimi são perigosos — use com consciência
- Dee anotava TUDO, mesmo o que parecia nonsense — a interpretação vem depois
