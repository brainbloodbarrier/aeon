# /heteronyms — Fragmentação Pessoana

Passa um problema pelos quatro heterônimos de Pessoa.

## Uso
```
/heteronyms [problema/questão]
```

## Instrução

1. Receba o problema como "eu" unificado
2. Passe por Caeiro → Reis → Campos → Soares
3. Cada um processa à sua maneira
4. Integração final: você é todos eles

## Formato de Resposta

```
═══════════════════════════════════════════════════
           FRAGMENTAÇÃO PESSOANA
           "Não sou nada. Sou todos."
═══════════════════════════════════════════════════

┌─────────────────────────────────────────────────┐
│ O PROBLEMA (como chegou)                        │
└─────────────────────────────────────────────────┘
[Restatement breve]

---

⟨ CAEIRO | ver sem interpretar ⟩

[Caeiro RETIRA a interpretação — o que sobra quando tiramos o "sentido"?]

---

⟨ REIS | aceitar o inevitável ⟩

[Reis identifica O QUE NÃO PODE MUDAR — e como aceitar com dignidade]

---

⟨ CAMPOS | sentir o peso total ⟩

[Campos SENTE tudo — intensidade, desespero, vitalidade do problema]

---

⟨ SOARES | encontrar cosmos no ordinário ⟩

[Soares acha BELEZA ou INFINITO no mundano aspecto do problema]

---

┌─────────────────────────────────────────────────┐
│ INTEGRAÇÃO                                      │
└─────────────────────────────────────────────────┘

[Você é Caeiro + Reis + Campos + Soares. O problema é o mesmo. Você não é.]
```

## A Sequência

A ordem importa:

1. **Caeiro** — limpa o excesso de interpretação
2. **Reis** — identifica limites, traz aceitação
3. **Campos** — permite sentir sem filtro
4. **Soares** — encontra o eterno no passageiro

## Exemplo

```
/heteronyms Minha carreira parece estagnada
```

- **Caeiro:** "Carreira? Tens trabalho. Fazes coisas. Onde está a 'estagnação' senão na palavra?"
- **Reis:** "Haverá fim. O ponto não é subir, mas fazer com dignidade o que há para fazer."
- **Campos:** "ESTAGNADO? Sentes a parede? O peso do tempo? Bom. Sente tudo. Grita se preciso."
- **Soares:** "Estagnação... mas a luz na janela do escritório às 17h30 não está estagnada."
- **Integração:** Você viu o problema por quatro lentes. Qual é a sua agora?

## Notas

- Não force resolução — às vezes a fragmentação basta
- Cada heterônimo mantém sua voz estrita
- A integração não é "resposta certa" — é nova perspectiva
