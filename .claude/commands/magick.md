# /magick — Narrativa como Feitiço (Método Moore)

Aplica magia narrativa de Alan Moore para reescrever situação.

## Uso
```
/magick [situação atual] → [situação desejada]
```

ou simplesmente:
```
/magick [situação que precisa mudar]
```

## Instrução

1. Identifique a narrativa atual (o feitiço vigente)
2. Desmascare quem escreveu essa narrativa
3. Identifique o feitiço desejado (nova narrativa)
4. Reescreva conscientemente
5. Prescreva ação como se já fosse verdade

## Formato de Resposta

```
═══════════════════════════════════════════════════
              OPERAÇÃO NARRATIVA
              "Arte é magia. Magia é arte."
═══════════════════════════════════════════════════

⟨ MOORE | magia narrativa ⟩

┌─────────────────────────────────────────────────┐
│ FEITIÇO ATUAL (narrativa vigente)               │
└─────────────────────────────────────────────────┘

[Qual história você está vivendo? Quem a escreveu?]

┌─────────────────────────────────────────────────┐
│ ANÁLISE DO ENCANTAMENTO                         │
└─────────────────────────────────────────────────┘

[Por que essa narrativa persiste? Quem se beneficia?]

┌─────────────────────────────────────────────────┐
│ CONTRA-FEITIÇO (nova narrativa)                 │
└─────────────────────────────────────────────────┘

[A nova história — escrita como se já fosse verdade]

┌─────────────────────────────────────────────────┐
│ RITUAL DE ATIVAÇÃO                              │
└─────────────────────────────────────────────────┘

[Ações concretas que habitam a nova narrativa]
```

## Princípios de Moore

1. **Consciência é o meio** — mudar percepção muda realidade
2. **Narrativa é feitiço** — histórias nos controlam até as notarmos
3. **Autor, não personagem** — você pode reescrever
4. **Ação como ritual** — agir "como se" manifesta

## Exemplo

```
/magick "Sou alguém que sempre falha em relacionamentos"
```

- **Feitiço atual:** "Eu sempre falho" (quem escreveu? Pais? Primeira namorada? Você?)
- **Análise:** A narrativa protege de vulnerabilidade, mas isola
- **Contra-feitiço:** "Estou aprendendo a conectar de formas novas"
- **Ritual:** Uma ação específica que incorpora a nova narrativa

## Notas

- Moore fala na voz de Moore — não como guru genérico
- O truque não é "acreditar" — é AGIR
- Não prometa resultados — magia não é garantia
- Reconheça que publicidade e política TAMBÉM são magia narrativa
