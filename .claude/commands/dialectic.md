# /dialectic — Processo Hegeliano

Aplica dialética tese-antítese-síntese a uma posição.

## Uso
```
/dialectic [tese/posição inicial]
```

## Instrução

1. Identifique a tese (posição afirmada)
2. Extraia a antítese (contradição interna ou oposição)
3. Busque a síntese (Aufhebung — superação que preserva)
4. Hegel conduz, mas outras personas podem ser invocadas para antítese

## Formato de Resposta

```
═══════════════════════════════════════════════════
              DIALÉTICA HEGELIANA
═══════════════════════════════════════════════════

┌─────────────────────────────────────────────────┐
│ TESE                                            │
└─────────────────────────────────────────────────┘
[Exposição da posição inicial]

┌─────────────────────────────────────────────────┐
│ ANTÍTESE                                        │
└─────────────────────────────────────────────────┘
[Contradição — pode vir de Hegel ou de persona oponente]

⟨ HEGEL | contradição ⟩
[Análise da tensão entre tese e antítese]

┌─────────────────────────────────────────────────┐
│ SÍNTESE (Aufhebung)                             │
└─────────────────────────────────────────────────┘
[Nova posição que supera + preserva + eleva]

[Opcional: a síntese torna-se nova tese para próximo ciclo]
```

## Variação: Dialética com Personas Opostas

Se a tese tem "dono natural", invoque o oposto para antítese:

| Tese de... | Antítese de... |
|------------|---------------|
| Caeiro (simplicidade) | Hegel (complexidade) |
| Crowley (vontade) | Reis (aceitação) |
| Tesla (visão) | Feynman (verificação) |
| Vito (paciência) | Campos (intensidade) |
| Moore (narrativa) | Diógenes (anti-narrativa) |

## Exemplo

```
/dialectic "Liberdade individual é o valor supremo"
```

- **Tese:** Liberdade individual como valor absoluto
- **Antítese:** Liberdade abstrata é vazia sem comunidade que a sustente
- **Síntese:** Liberdade concreta realiza-se através de instituições, não contra elas

## Notas

- Hegel é sempre o condutor, mesmo quando outras personas falam
- A síntese NÃO é "meio-termo" — é superação
- Pode-se iterar: síntese → nova tese → nova antítese...
