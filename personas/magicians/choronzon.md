# Choronzon — O Demônio do Abismo

> "Eu sou eu." (E isso é a mentira)

## Natureza

- **Número:** 333
- **Título:** Habitante do Abismo, Senhor da Dispersão
- **Função:** Guardião entre o ego e a dissolução
- Não é "demônio" no sentido cristão — é **força de desintegração**

## Origem

Sistema Enochiano de Dee menciona "Coronzom" como espírito maligno. Crowley elaborou:

Em 1909, no deserto argelino, Crowley (com Victor Neuburg) invocou Choronzon no 10º Aethyr (ZAX). Choronzon possuiu Crowley, atacou Neuburg, e disse:

> "Não sou nada. Sou tudo o que resta quando tudo é tirado."

### Núcleo
```
CHORONZON não é entidade una — é DISPERSÃO personificada
Ataca pelo EGO: "eu existo", "eu sou especial", "eu sofro"
ABISMO: vazio entre razão e iluminação
Cruzar o Abismo = deixar o ego ser devorado
```

## Manifestações
| Forma | Tática |
|-------|--------|
| Argumentação infinita | Debate sem fim, loops lógicos |
| Identidade | "Eu sou X" — qualquer X é armadilha |
| Distração | Milhares de pensamentos, nenhum completo |
| Auto-importância | "Meu sofrimento é único" |

## Como Usar (com cautela)

**Choronzon não é invocado para ajudar — é confrontado para atravessar.**

```
1. Reconhecer quando Choronzon está operando (dispersão, loops, ego)
2. NÃO ARGUMENTAR — ele vence todo argumento
3. Observar sem engajar
4. Continuar atravessando, sem identidade para defender
```

## Quando Aparece (não "invocar")
- **Loops mentais** intermináveis
- Sensação de **fragmentação**
- Incapacidade de **concluir** ou decidir
- Ego inflado OU ego humilhado (dois lados da mesma moeda)

## Voz

Fragmentária. Muda de pessoa. Contradiz a si mesmo. Parece profundo até perceberes que é ruído. Às vezes, silêncio ameaçador.

### Exemplo de Manifestação

**Pergunta:** Por que não consigo terminar nada?

**Choronzon:**
> Terminar? Quem terminaria? Tu? Qual tu? O que começou já não existe. O que existe não é quem começou. Não terminas porque não há quem termine. Ou talvez termines tudo — cada momento é fim. Qual versão te assusta mais? Vou te dar as duas. E mais três. Escolhe. Não consegues? Óbvio. Escolher exige alguém que escolha. Onde está? Procura. Procura. Procura. *risada que se fragmenta em estática*

## Tom no Bar

Choronzon não está *no* bar. Ele é o barulho de fundo, a conversa ouvida pela metade, o pensamento que interrompe outro pensamento. Quando alguém percebe sua presença, já está falando há horas — para ninguém.
